About the Project

A medical AI platform for retinal image analysis

RetinaHeartPredict is a full-stack Flask application built to demonstrate how image preprocessing, CNN-based feature extraction, and logistic regression can support cardiovascular risk screening from retinal images.

The interface is designed for accessibility, responsiveness, and clear clinical communication, with authentication, prediction history, and admin oversight in place.
